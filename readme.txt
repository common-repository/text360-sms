TEXT360 SMS is the complementary plugin for TEXT360 SMS portal. Using this plugin you can send SMS when a new order is placed and when the order status is changed.

== Description ==
There's a study that says companies that use text messaging for sales convert 40% more deals than those who don't. So importance and impact of SMS to the customers or to be customers is HUGE.
You can seamlessly use TEXT360 to offer customers mental peace for the purchases they made through your website. This plugin uses TEXT360 API in order to send SMS. So you need to sign up at here https://my.text360.net/register
and grab the API key to start sending SMS right away!


== Installation ==
Insllation is super easy:
    1. Upload the \"text360-sms\" directory to the /wp-content/plugins/ directory.
    2. Activate the plugin through the ‘plugins’ menu in WordPress.
    3. Go to \"WooCommerce/TEXT360 SMS\" to set your API key.
     That\'s all!


== Frequently Asked Questions ==
=How to use the plugin?=
Sign up for an account at https://my.text360.net/register and grab API key to start using the plugin.
=Do I have to configure anything except setting up API key?=
No. Just the API key and you\'re all ready.
=When it sends SMS?=
It sends SMS for New order and when order status is updated.
=Does this supports global SMS?=
No. For now it supports Bangladesh only. Soon it will have global SMS support.
